# Sravni Tool Ultimate 📋

**Sravni** — это простой и эффективный инструмент для сравнения списков данных и подготовки SQL-запросов. 

---

## 🚀 Как пользоваться инструментом

Вы можете использовать инструмент двумя способами:

### 1. Веб-версия (Работает в браузере, установка не нужна)
Если вы хотите просто сравнить данные прямо сейчас:
1.  Нажмите на файл [**index.html**](index.html) в списке выше.
2.  В открывшемся окне кода нажмите кнопку **Download raw content** (значок скачивания справа сверху) или просто скачайте весь архив проекта через кнопку **<> Code -> Download ZIP**.
3.  Откройте скачанный файл `index.html` двойным кликом в любом браузере (Chrome, Edge, Firefox).

> [!TIP]
> **Скоро здесь будет прямая ссылка**, по которой инструмент будет открываться сразу в окне браузера без скачивания!

---

### 2. Desktop-версия для Windows (Готовый .exe файл)
Если вам привычнее работать в отдельном окне программы:
1.  Перейдите в раздел [**Releases**](https://github.com/Andrey13131/SravniWeb/releases) (справа в колонке на главной странице репозитория).
2.  Скачайте файл `Sravni.exe`.
3.  Запустите его. Программа не требует установки.

---

## ✨ Что умеет Sravni?

*   **🔍 Поиск отличий**: Сравните два списка и мгновенно получите список строк, которых не хватает во втором окне.
*   **💾 SQL Helper**: Нажмите одну кнопку, и ваш список превратится в готовый фрагмент SQL-кода: `'строка1', 'строка2'`.
*   **🔢 Умная нумерация**: Видьте точное количество строк в каждом окне.
*   **🧹 Быстрая очистка**: Управляйте данными одним кликом.

---

## 🛠 Техническая информация
Инструмент написан на **Python** (Desktop) и **Vanilla JS** (Web). Он полностью автономен и не передает ваши данные на внешние серверы — всё происходит локально на вашем компьютере.
