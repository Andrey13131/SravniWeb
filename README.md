# Sravni Tool Ultimate 📋

**Sravni** — это мощный и простой инструмент для сравнения текстовых данных, специально разработанный для быстрой подготовки SQL-скриптов и анализа расхождений в таблицах.

![Sravni Web Preview](https://via.placeholder.com/1200x600/0078D7/FFFFFF?text=Sravni+Tool+Web+Edition) 
*(Дизайн в стиле Windows 11 / Slate Dark)*

## ✨ Основные возможности

- **Сравнение данных**: Мгновенный поиск строк, которые есть в первом списке, но отсутствуют во втором.
- **SQL Форматирование**: Автоматическое преобразование списка строк в формат `'item1', 'item2', ...` для вставки в `WHERE IN (...)`.
- **Живая нумерация**: Удобная нумерация строк в реальном времени.
- **Две версии**: 
    - 🖥️ **Desktop (Python)**: Мощное приложение для Windows.
    - 🌐 **Web (HTML)**: Автономная версия для работы в браузере без установки.

## 🚀 Как начать

### Веб-версия (Самый быстрый способ)
1. Откройте файл `index.html` в любом современном браузере (Chrome, Edge, Firefox).
2. Пользуйтесь! Не требует установки и интернета.

### Desktop-версия
1. Скачайте `Sravni.py`.
2. Убедитесь, что у вас установлен **Python 3.10+**.
3. Запустите файл командой:
   ```bash
   python Sravni.py
   ```
   *Или используйте готовый `.exe` файл в разделе Releases.*

## 🛠 Технологии

- **Python & Tkinter**: Для настольной версии.
- **HTML5 / CSS3 / JS**: Для современной веб-версии (без внешних библиотек).
- **PyInstaller**: Для сборки независимых исполняемых файлов.
